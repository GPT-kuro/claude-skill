# Vision

AIを使った情報発信を、もっと簡単にする。

AIニュース収集・要約・SNS投稿・記事作成など、
情報発信者が日常的に行う作業を自動化し、
「発信のOS」として機能するツールを目指す。

## Problem

AI情報発信者は以下の課題を抱えている。

- AIニュースを追うのに時間がかかる
- 投稿ネタを考えるのが大変
- X投稿・note記事作成に時間がかかる
- 投稿管理がバラバラ

## Solution

AI Creator OS

AIニュース → 要約 → SNS投稿 → 記事化

この流れを1つのシステムで自動化する。

## Goal

AI情報発信の作業時間を  
1/5に削減する。
